This page details compatibility between OpenTofu and Terraform. If you see a feature that you'd like to see supported, please check the [OpenTofu issues](https://github.com/opentofu/opentofu/issues).

## Core

| Feature                                 | OpenTofu           | Terraform |
|-----------------------------------------|--------------------|-----------|
| [`tofu test`](/docs/cli/commands/test/) | 1.6.0              | 1.6.0     |
| `removed` block                         | 1.7.0 (unreleased) | 1.7.0     |
| Use `for_each` to expand `import`       | 1.7.0 (unreleased) | 1.7.0     |
